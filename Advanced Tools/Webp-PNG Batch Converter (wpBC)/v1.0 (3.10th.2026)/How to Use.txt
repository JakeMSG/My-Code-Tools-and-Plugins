==== 1. Type: "shell:sendto" into the path of the explorer window
==== 2. Make a shortcut to the Bat files in this folder
==== 3. Right-click any folder (even with subfolders!), then click "Send To", and choose the Bat file you want to use
** Of course, you can also just drag the Files/Folders needed directly onto the Batch files, but the way I described makes the process much easier


==== For the "PNG_to_WebP" conversion, you're first prompted for the quality of the conversion, between 0 and 100 (%). Default (if you simply press Enter instead of entering a number first) is 80 (%).
